# tower_defense

Tower defense game in python

## Project Structure
tower_defense/
├── main.py
├── game.py
├── tower.py

## Installation 
1. Clone the repository:
```bash
git clone https://github.com/yahao333/tower_defense
```
2. Install the required dependencies:
```bash
pip install pygame
```
